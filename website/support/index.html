Support

Help for setup, download, and privacy questions.

Yemma 4 is an on-device AI app. If something is broken, if the model download stalls, or if you need clarification for App Review, use the contact options below.

Quick contact

Email
justin@avmillabs.com
GitHub
github.com/jskoiz/yemma-4/issues

What to include

  • Your device model and iOS version.
  • Whether you are using the initial model download or an already downloaded model.
  • The exact screen, button, or message where the issue happens.
  • If relevant, a screenshot or short screen recording.

Common questions

  • Does Yemma send my chats to a third-party AI service? No. Chat input and output are processed locally on the device.
  • Why does the app download a model? The first launch downloads the model files needed for on-device inference.
  • Can I delete my local data? Yes. You can delete downloaded models and conversation history from inside the app.

App Review note

For review questions, use the same email above and mention the submission ID from App Store Connect so we can respond quickly.
